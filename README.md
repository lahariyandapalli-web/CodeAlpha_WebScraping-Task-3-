# CodeAlpha_WebScraping-Task-3-
Data Visualization using Tableau
Project Title:
Data Visualization using Tableau

Description:
This project focuses on transforming raw data into meaningful visual representations using Tableau. The dataset obtained from web scraping was used to create clear and insightful charts and dashboards for better understanding of patterns and trends.

Dataset Used:
Dataset collected from http://quotes.toscrape.com (Task 1 output)

Tools Used:

- Tableau

Data Visualized:

- Quotes
- Authors

Visualization Techniques:

- Created bar charts to represent number of quotes per author
- Applied sorting and filtering for better clarity
- Designed dashboards to present data in an organized manner
- Used labels and formatting to enhance readability

Storytelling & Insights:

- Visualizations clearly show that some authors have more quotes than others
- The data distribution is uneven across authors
- Dashboards help in quickly understanding key patterns and trends

Output:

- Tableau dashboard (.twbx file)
- Charts and visualizations

Conclusion:
Successfully transformed raw data into meaningful visual insights using Tableau. The visualizations improved data understanding and helped in identifying key patterns, making the data more useful for decision-making.
